# Визуальные требования (FitMetrics)

## 1. Общий тон
- Спокойный, минималистичный интерфейс.
- Акцент на «воздухе» и читаемости.
- Минимум визуального шума.

## 2. Типографика
- Основной шрифт: Sans Serif (читаемый, без засечек).
- Иерархия: H1 / H2 / Body / Caption с заметной разницей кегля и интервалов.
- Межстрочный интервал: комфортный для сканирования данных.

## 3. Графический язык
- Большие радиусы: 16–24 px для карточек и кнопок.
- Тени: минимальные, мягкие, либо тонкая граница 1 px.
- Иконки: линейные (outline), тонкое начертание.
- Индикаторы прогресса: кольцевые диаграммы, мягкие градиенты или тонкие линии.

## 4. Компоненты
- Кнопки: мягкие формы, без агрессивных заливок.
- Карточки: светлый фон, 1 px обводка, минимум теней.
- Поля ввода: чистая геометрия, ясные состояния.

## 5. UI-kit (Delivery)
- Style Guide: палитра (HEX/RGB), шрифтовые пары, отступы (Grid 8 px).
- Компоненты: состояния кнопок (Default/Pressed/Disabled), поля ввода, чекбоксы, переключатели.
- Экраны в Light/Dark mode:
  - Главный экран (Дашборд).
  - Дневник питания + карточка продукта.
  - Экран тренировки (Таймер/Упражнение).
  - Профиль и графики прогресса.

## 6. Анти-референсы (не использовать)
- Неоновые/кричащие цвета.
- Острые углы.
- Скевоморфизм и перегруженные детали.
- Сложные игровые анимации (конфетти и т.п.).
